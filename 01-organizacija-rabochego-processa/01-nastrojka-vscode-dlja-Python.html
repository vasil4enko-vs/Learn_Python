Настройка Visual Studio Code для программирования на Python

Установка редактора Visual Studio Code

Большинство опытных программистов рекомендуют использовать новичкам для обучения программированию на Python либо такие IDE как IDLE и Thonny, либо такую мощную IDE как PyCharm от компании JetBrains

Но лично мне очень понравился Visual Studio Code. Мощный редактор, который при помощи плагинов можно превратить в достойного конкурента не дёшево стоящего PyCharm.

Справедливости ради стоит указать, что PyCharm Free Community можно использовать совершенно бесплатно, но только для разработки на чистом Python.

Установите языковой пакет, чтобы изменить язык отображения на русский. (Install language pack to change the display language to Russian.)
