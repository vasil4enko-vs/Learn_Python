# Ответ к задаче 4.2.4

```python
num_day = int(input())
day = ["понедельник", "вторник", "среда", "четверг", "пятница", "суббота", "воскресенье"]

if 1 <= num_day <= 7:
    print(day[num_day - 1])
elif num_day < 0:
    print("Номер дня недели не может быть отрицательным.")
else:
    print("Номер дня недели не может быть больше '7'.")
```
