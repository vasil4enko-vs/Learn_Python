# Задача 3.9.4

Имеется вложенный список:

```python
a = [True, [1, 0, ["True", ["Истина", "Ложь"], "False"]], False]
```

Как записать индексы, чтобы обратиться к элементу со значением "Истина"?

Выберите один вариант из списка

- `a[1][2][1][0]`
- `a[1][2][0][1]`
- `a[1][2][1]`
- `a[1][2][0]`
- `a[2][3][2][1]`
