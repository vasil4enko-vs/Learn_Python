# Ответ к задаче 3.9.6

```python
t = [["Скажи-ка", "дядя", "ведь", "не", "даром"],
     ["Я", "Python", "выучил", "с", "каналом"],
     ["Балакирев", "что", "раздавал?"]]

print(word in t[0] or word in t[1] or word in t[2])
```
