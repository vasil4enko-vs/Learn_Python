# Ответ к задаче 3.4.1

| Спецсимвол | Значение спецсимвола                           |
|:----------:|------------------------------------------------|
| \n         | Перевод строки                               |
| \\         | Символ обратного слэша                       |
| \’         | Символ апострофа (одинарной кавычки)         |
| \"         | Символ двойной кавычки                       |
| \b         | Эмуляция клавиши BackSpace                   |
| \r         | Возврат каретки                              |
| \t         | Горизонтальная табуляция                     |
