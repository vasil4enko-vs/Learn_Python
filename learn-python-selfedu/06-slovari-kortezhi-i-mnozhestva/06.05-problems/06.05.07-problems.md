# Задача 6.5.7

Необходимо прочитать это число и разложить его на простые множители. Затем, проверить, содержит ли оно множители 2, 3 и 5 (обязательно все их, хотя бы один раз). Если это так, то вывести "ДА", иначе "НЕТ".
