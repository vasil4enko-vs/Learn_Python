Сопоставьте значения из двух списков

| Оператор | Значение оператора         |
|:--------:|----------------------------|
| <        | сравнение меньше или равно |
| >        | сравнение больше или равно |
| <=       | сравнение на неравенство   |
| >=       | сравнение на равенство     |
| ==       | сравнение на больше        |
| !=       | сравнение на меньше        |
