Выбор редактора, IDE для программирования на Python

Большинство опытных программистов рекомендуют использовать новичкам для обучения программированию на Python либо такие инструменты как IDLE и Thonny, либо такую мощную IDE как PyCharm от компании JetBrains.

Но я думаю, что IDLE и Thonny слишком аскетичны как для полноценных IDE, а PyCharm чересчур "перекачан", в нём есть сразу всё и многое из этого может и не понадобиться.

К тому же бесплатную версию PyCharm Community, можно использовать только для разработки на чистом Python. Если же Вам будет нужна поддержка HTML, JavaScript и SQL, то Вам таки придётся купить PyCharm Professional. А это довольно недёшево.

Есть конечно ещё много других IDE, такие как Eclipse, Eric, Geany, Spyder... И Вы вольны выбрать себе тот инструмент, который Вам нравится. В принципе, это дело вкуса.

Мне же приглянулся Visual Studio Code. Лёгкий редактор, который при помощи плагинов можно превратить в мощную IDE.

Чем мы и займёмся далее.
