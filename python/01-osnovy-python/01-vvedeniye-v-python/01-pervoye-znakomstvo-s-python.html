Первое знакомство с Python

Для начала определимся, как произносится название языка программирования Python. В русском языке его обычно называют Пайтон или Питон.

Ну, а теперь, разберёмся, что же это за зверь такой - Python. Python - высокоуровневый язык программирования общего назначения с динамической строгой типизацией и автоматическим управлением памятью, ориентированный на повышение производительности разработчика, читаемости кода и его качества, а также на обеспечение переносимости написанных на нём программ.
