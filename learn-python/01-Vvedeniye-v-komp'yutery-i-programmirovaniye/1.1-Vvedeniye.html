1. Введение в компьютеры и программирование

1.1. Введение

В наше время компьютеры так или иначе используются почти во всех сферах деятельности человека. Так ПК используются для учёбы, работы и отдыха. Специализированные компьютеры используются во множестве разных устройств от утюгов до космических аппаратов. Также, по сути своей, компьютерами являются такие универсальные устройства как смартфоны и планшеты.

Что же делает все эти компьютеры такими популярными?

Конечно, возможность их запрограммировать под разные задачи.

А программируют все эти устройства с помощью различных языков программирования. И одним из таких языков является Python.

Python - это высокоуровневый язык программирования общего назначения с динамической строгой типизацией и автоматическим управлением памятью, ориентированный на повышение производительности разработчика, читаемости кода и его качества, а также на обеспечение переносимости написанных на нём программ. Подробнее см. здесь.

Python является одним из самых популярных языков программирования. Два его основных преимущества - простота и универсальность.

Этот язык действительно прост. Выучить его гораздо проще чем большинство других языков программирования, его часто изучают как первый язык программирования и используют для обучения детей.

Простота же, в свою очередь, позволяет программисту сосредоточится именно на выполнении задачи, а не на синтаксисе языка.

В тоже время, Python - это мощный и универсальный язык.

Он широко используется в веб-разработке (Django, Flask и др. фреймворки), для тестирования аппаратного обеспечения (такие компании как Intel, Cisco, IBM и др.). Также Python используется для разработки десктопных и мобильных приложений, игр, в анализе данных, машинном обучении, образовательных проектах и т.д.

Будучи языком общего назначения Python применим почти во всех сферах деятельности. Фактически Python, тем или иным образом, используют практически все солидные IT-компании.

Python, в настоящее время, используется во многих сферах и его популярность неуклонно растёт.

Также необходимо знать, что всего у Python было три основные версии:

  • Python 1
  • Python 2
  • Python 3

Python 3 нарушил обратную совместимость с предыдущими версиями Python. И так как разработчики прекратили поддержку предыдущей версии Python 2 в 2020 году. То нами будет рассматриваться только Python 3 .
