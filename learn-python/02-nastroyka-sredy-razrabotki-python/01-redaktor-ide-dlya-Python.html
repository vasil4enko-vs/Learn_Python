Выбор и установка IDE для программирования на Python

Выбор IDE

Большинство опытных программистов рекомендуют использовать новичкам для обучения программированию на Python либо такие инструменты как IDLE и Thonny , либо такую мощную IDE как PyCharm от компании JetBrains .

Но я думаю, что IDLE и Thonny слишком аскетичны как для полноценных IDE, а PyCharm чересчур "перекачан", в нём есть сразу всё и многое из этого попросту может не понадобиться.

К тому же бесплатную версию PyCharm Community , можно использовать только для разработки на чистом Python. Если же Вам будет нужна поддержка HTML, JavaScript и SQL, то Вам придётся купить PyCharm Professional. А это довольно недёшево .

Есть конечно ещё много других IDE, такие как Eclipse, Eric, Geany, Spyder... И Вы вольны выбрать себе тот инструмент, который Вам нравится. В принципе, это дело вкуса.

Мне же приглянулся Visual Studio Code разработанный компанией Microsoft . Это сравнительно лёгкий редактор, который при помощи плагинов можно превратить в мощную IDE.

Чем мы и займёмся далее.

Установка Visual Studio Code

Установка Visual Studio Code напрямую зависит от операционной системы, стоящей на Вашем ПК.

Если Вы, как и я, являетесь пользователем Linux , то для установки VS Code Вам необходимо ознакомиться со следующей страничкой документации .

Почитатели macOS могут ознакомиться с процессом установки VS Code здесь .

А любителям Windows - сюда .

После установки VS Code посмотрим на него поближе.

При первом запуске VS Code будет иметь примерно такой вид:

Окно VS Code при первом запуске.

Обратите внимание на уведомление открывшееся в правом нижнем углу со следующим текстом:

Установите языковой пакет, чтобы изменить язык отображения на русский. (Install language pack to change the display language to Russian.)

Если Вы пожелаете изменить язык интерфейса редактора на установленный в Вашей системе, то нажмите на кнопку в этом уведомлении - "Установить и перезапустить (Install and Restart)".

После перегрузки VS Code, поздравляем себя, Вы установили первый плагин Russian Language Pack for Visual Studio Code - языковой пакет для русского языка. А интерфейс VS Code стал теперь русскоязычным.

Примечание: Если в Вашей операционной системе установлен по умолчанию язык отличный от русского, то к установке будет предложен соответствующий языковой пакет.

Ну, а в том случае, когда Вам по какой-либо причине VS Code не предложит установку языкового пакета, его можно будет установить точно так как любое другое расширение.

И как это сделать, мы скоро узнаем.
