Другие расширения VS Code устанавливаются точно также.

Примечание: После установки расширения Python (Microsoft) можно обнаружить что это расширение установилось не одно. По зависимостям установятся ещё два расширения:

  • Pylance (Microsoft)
  • Jupyter (Microsoft)

В свою очередь расширение Jupyter может предложить установку или установить ещё два расширения:

  • Jupyter Keymap
  • Jupyter Notebook Renderers
